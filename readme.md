# node-address

## Dependencies:

```
node v14.16.0
npm v6.14.11
```

## Run the project:

To run docker, just run `docker-compose up` and the start script will run until the tests.

To run locally, use `npm run test` to run the tests.

Do not forget to put down environment variable such as :
```
VIA_CEP
GOOGLE_MAPS_URL
GOOGLE_MAPS_API_KEY
IBGE_URL
```
